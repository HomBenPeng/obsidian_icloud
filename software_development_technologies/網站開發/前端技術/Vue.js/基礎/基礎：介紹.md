## 什麼是 Vue.js？
是一款用於構建用戶界面的 JavaScript 框架。它基於標準 HTML、CSS 和 JavaScript 構建，並提供了一套聲明式的、組件化的編程模型，幫助你高效地開發用戶界面。無論是簡單還是複雜的界面，Vue 都可以勝任。

## Vue.js 的核心概念
1. 應用程式
2. 組件
3. 模板語法
4. 狀態管理
5.  生命週期鉤子
